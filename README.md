# dotnet-zero  

## Dev setup
- NodeJS 12.x with latest npm
- .NET Core SDK 3.1 - latest preview
- VS Code with the following extensions: C#, C# Extensions, vscode-icons, YAML

## Running the project
In the dotnet-zero-engine folder:    
dotnet watch build  
In the dotnet-zero-ui folder:  
npm start  
Visit http://localhost:8080
